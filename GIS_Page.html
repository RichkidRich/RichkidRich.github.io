QuikTour: Geographical Information System (GIS) for Tourists written in C++

Project Description

QuikTour is a geographical information system created for tourists to easily generate a trip itinerary based only on the location they are visiting and the duration of their visit.

Tools and Skills

  • C++
  • CSS
  • Git

Detailed Overview

As part of the second year curriculum at the University of Toronto, Computer Engineering students were told to form teams of three and create a mapping tool over the course of the semester in C++ using the OpenStreetMap API. The map had to include the basic functions of mapping tools such as the ability to search different street names. We also added our own additional features to enhance the map as much as possible. Some of these features include:

  • Tutorials for first time users
  • Color blind mode support

This project also included two presentations. The first was a progress update which also included a feature showcase. It was important for us to be able to market the features we already had but also planned on implementing as feasible to the audience. The second presentation was purely a showcase of our final product, as well as a pitch for the future implementation of the GIS.

Our team pitched QuikTour, an extension of the mapping tool we had already made. QuikTour would mainly utilise the pathfinding algorithm we wrote for the traveling salesman problem, and apply it to the realm of tourism. A tourist on a layover simply inputs the duration of their visit and QuikTour will optimize a path for visiting attractions at their location. A detailed set of directions would also be displayed, allowing the tourist to navigate what is likely an unfamiliar foreign city.

This project undoubtedly requires good programming skill to complete however the most important factor at the end of the day was the teams ability to work together, communicate concerns and resolve issues. Our team was able to set up a strong communication and task assignment eco system, allowing us to work with each other much easier.
